Кодирование и декодирование текста с помощью алгоритма RLE можно выполнить с помощью командной строки и Node.js.

Для запуска программы необходимо:

Установить Node.js на свой компьютер, если он еще не установлен.
Создать два текстовых файла: исходный текстовый файл и файл, в который будет записан результат работы программы. Имена файлов можно выбрать произвольно, но расширение должно быть ".txt".
Открыть командную строку.
В командной строке перейти в директорию, где находятся файлы программы и текстовые файлы.
Ввести команду в формате:
node is_it_rle.js <mode> <input_filename> <output_filename>
где:

mode - режим работы программы. Возможны два значения: code (кодирование) и decode (декодирование).
input_filename - имя файла с исходным текстом, который нужно закодировать или раскодировать.
output_filename - имя файла, в который будет записан результат работы программы.
Например, чтобы закодировать файл input.txt и сохранить результат в файл output.txt, необходимо ввести следующую команду:
node is_it_rle.js code input.txt output.txt

Аналогично, чтобы раскодировать файл output.txt и сохранить результат в файл input_decoded.txt, необходимо ввести следующую команду:
node is_it_rle.js decode output.txt input_decoded.txt

Если в процессе работы программы возникает ошибка, программа выведет сообщение об ошибке в консоль, а также завершит свою работу.

Обратите внимание, что в именах файлов и режиме работы программы необходимо указывать корректные значения, иначе программа не сможет выполнить свою работу.
