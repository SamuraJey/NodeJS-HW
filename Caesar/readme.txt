Реализация дешифратора шифра Цезаря на языке JavaScript.

- Для запуска js файла никаких дополнительных параметров не нужно. 
- Программа предназанчена для работы со строками на английском языке.
- В папке с программой должен лежать файл alphabet.txt с частотами букв латинского алфавита.
- Программа считывает исходную строку из файла input.txt и выводит результат дешифрования
  в файл output.txt.
