
# Шифр цезаря

Программа умеет кодировать с заданным сдвигом и декодировать на основе частотного анализа шифр Цезаря. Прогамма читает файл alphabet.txt в котором должны содержаться частотности букв английского алфавита. Код чистый и красивый и понятный, есть комментарии, разберетесь \
**Функции**:
* -e кодировать текст и -s указать сдивг шифровки
* -d декодировать


## Примеры использования

Закодировать шифром цезаря (-e), файл, который надо закодировать, указать сдвиг (-s X). Если свдиг не указан, выбирается случайный
```javascript
node Caesar.js -e file.txt -s 5
```
Декодировать, программа сама подбирает сдвиг для декодировки, на основе частотного анализа
```javascript
node Caesar.js -d encryptedfile.txt
```
